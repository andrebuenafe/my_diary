## Duty Diary

Duty Diary is a web application for the Diary for the Interns and manage the trainee by admin which has supervisor that can approve all the diaries which trainees wroted day by day.

## Steps to run this system after cloning

-   run composer install
-   run cp .env.example. .env
-   run php artisan migrate
-   run php artisan db:seed --class=RolesSeeder
-   run php artisan db:seed --class=AdminSeeder
-   run npm install
-   run npm run dev
-   php artisan optimize
-   php artisan serve
